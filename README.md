### spring-boot-sample

#### 项目说明
SpringBoot 2.1.x 整合其他框架：Mybatis、Dubbo、Swagger2等

#### Maven模块描述

| 端口 | 模块名称 | 描述 |
| --- | --- | --- |
| ----- | sample-api | 服务接口、工具类、实体类 |
| 10081 | sample-business | 业务接口 |
| ----- | sample-dal | mapper、sql语句 |
| 20881 | sample-server | 服务接口实现 |
